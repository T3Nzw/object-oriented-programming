# Задачи

## Задача 00

Нека е даден следният клас:

```c++
class A {
public:
  A() { std::cout << "A()\n"; }
  A(int) { std::cout << "A(int)\n"; }
  A(A const &) { std::cout << "A(A const &)\n"; }
  A &operator=(A const &) { std::cout << "A &operator=(A const &)\n"; return *this; }
  ~A() { std::cout << "~A()\n"; }

  int getX() const { return x; }

private:
  int x;
};
```

До какво поведение биха довели следните редове код:

```c++
  {
1.  A a1, a2(4);
2.  A a3(a1);
3.  A a4 = a2;
4.  a1 = a3 = a4;
5.  A const a5 = a3 = a2;
6.  a4 = 5;
7.  a2 = 'a';
8.  a5 = a1;
  }
```

```c++
  {
1.  A a1;
2.  A* p1;
3.  p1 = &a1;
4.  A* p2 = new A;
5.  A a2 = *p2;
6.  delete p1;
7.  delete p2;
8.  std::cout << a2.getX();
  }
```

## Задача 01

Да се напише клас `String`, представящ низ с произволна големина. Класът да поддържа следния интерфейс:

- възможност за създаване на празен низ;
- възможност за създаване на низ по подаден C-style низ;
- проверка дали даден низ е празният такъв;
- връщане дължината на низа;
- връщане C-style низа, който се съдържа в обекта;
- извеждане на съдържанието на низа на стандартния изход;
- добавяне на един символ на края на низ;
- конкатенация на два низа;
- премахване на последния символ в низа, ако такъв има;
- обръщане реда на символите в низа;
- изчистване на съдържанието в низа;
- изтриване на `n` на брой последователни символа, започвайки от някакъв индекс `i`;
- проверка дали даден низ се съдържа в друг. Ако това е изпълнено, да се върне индекса,
  на който започва първото срещане на подниза;
- възможност за създаване на нов низ, който представлява "отрез" от друг низ.
